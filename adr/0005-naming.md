# ADR-0005: Naming: Argelander proper, AGE functional

Status: accepted. Date: 2026-07-10.

## Decision

The component proper name is Argelander; AGE remains the functional identity (requirement IDs, strip schema, spec references). npm names argelander, argelander-core, argelander-leaflet, argelander-three claimed by placeholder publish; GitHub handle argelander registered.

## Rationale

Friedrich Argelander was Bessel's pupil and methodological heir; the Bonner Durchmusterung's drift-scan zone survey is the historical form of pushbroom strip coverage accumulated into an atlas. Registry sweep 2026-07-10: all names free on npm, PyPI, crates.io; GitHub handle unregistered. Functional lineage: footprint path prediction descends from Besselian eclipse elements.

## Addendum (2026-07-11): registry state corrected

The placeholder publish recorded above never executed: a registry check on 2026-07-11 found all four names unpublished, and argelander-providers (added to the claim set by ADR-0008) equally free. All five names were re-verified free the same day. The publish requires interactive npm authentication and is deferred to the repository owner; until it lands, the npm claim in this record is intent, not registry fact. Publish argelander-core first, since every other placeholder declares it as a runtime dependency. The GitHub handle registration stands as recorded.
