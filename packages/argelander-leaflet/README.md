# argelander-leaflet

Leaflet adapter for argelander-core strips (ADR-0003): `AcquisitionLayer` paints strips into a canvas overlay through pure, headless-tested modules. The six treatments of AGE-07 (outline, flat fill, now plus trail, mechanism texture, quality gradient, time gradient) are styling policies over the strip schema, selectable per layer at runtime; hue is reserved for acquisition state with the atlas palette by default and instrument identity rides a dash-pattern texture, both explicitly overridable (AGE-08). Level of detail switches between envelope and mechanism by projected swath width and sparse geometries are never inflated to ribbons (AGE-09). Segments never ribbon across time gaps, burst seams, or frame chains (SPEC-STRIP section 2). Antimeridian crossings paint per world copy from per-quad unwrapped longitudes, and the projector is Leaflet's own `latLngToContainerPoint`, so MMGIS CRS configurations, polar included, flow through untouched (AGE-10). The now-trail treatment extrudes committed coverage into a persistent trail canvas with exponential decay; animation is pausable and speed-scalable (AGE-16). Leaflet is a peer dependency: the host map brings its own.
